Servlet Programs 

Step to Create Dynamic Web Project
1. Go to File → New → Dynamic Web Project
(If not visible or available, go to File → New → Other → Web → Dynamic Web Project)

2. Project Configuration:
   Project name: Enter your project name (e.g., SampleParameterServlet)
   Target runtime: Select Apache Tomcat (configured in Step 1)
   Dynamic web module version: 4.0 or 6.0 (recommended) based on Tomcat Server
   Configuration: Default Configuration

3. Click Next (twice) 

4. Web Module Settings:
   Check Generate web.xml deployment descriptor
   Click Finish


Step for Creating Servlet Manually

1. Right-click on created dynamic project or src/main/java folder within project
2. Select New → Class
   Enter:
	Package: cseai.cse (optional)
	Name: SampleParameterServlet (or any name)
	Superclass: javax.servlet.http.HttpServlet (based on tomcat server)

   Click Finish
